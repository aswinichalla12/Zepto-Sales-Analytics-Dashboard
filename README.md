# Zepto-Sales-Analytics-Dashboard
Built an interactive Zepto Sales Dashboard in Microsoft Excel to visualize key business metrics, analyze sales trends, and generate actionable insights through dynamic reports.

📊 Zepto Sales Analytics Dashboard
📌 Project Overview

The Zepto Sales Analytics Dashboard is an interactive Microsoft Excel project designed to analyze and visualize sales performance. It provides meaningful business insights through KPI cards, Pivot Tables, Pivot Charts, Slicers, and dynamic visualizations, helping users understand sales trends and make data-driven decisions.

🎯 Objectives
Analyze sales performance and business KPIs.
Identify top-performing products and categories.
Monitor sales trends over time.
Compare sales across different cities and payment methods.
Build an interactive dashboard for business reporting.
🛠️ Tools & Technologies
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Conditional Formatting
Excel Formulas
📂 Dataset

The project uses a Zepto sales dataset containing order details, product information, customer data, sales, profit, quantity, payment mode, and regional sales information.

📈 Dashboard Features
Total Sales, Orders & Profit KPIs
Monthly Sales Trend Analysis
Category-wise Sales Analysis
Product Performance Analysis
City-wise Sales Analysis
Payment Mode Analysis
Interactive Slicers for Dynamic Filtering
